# mental-health-cycle-tracker


**##Mental Health & Cycle Tracker A machine learning project analyzing the connection between menstrual cycle phases and anxiety patterns using real participant data.
A Telegram chatbot that collects menstrual cycle , sleeep exerise , anxiety , triggers  , and coping method data . Responses are stored in GOgogle Sheets and analyzed using Python , data identify patterns and provide personalized wellness insights.
**

**##Features**
-Telegram chatbot for data collection
-Google Sheets integration for real-time storage
-Menstrual cycle tracking
-anxiety score trackring
-Anxiety triggers and coping method analysis 
-data cleaning and preprocessing 
-Exploratory Data analysis(EDA)
-Machine Learning-based pattern analysis
-Personalized wellness insights


**##Tech Stack**
-Python
-telegram Bot API
-Google sheets API
-Pandas
-Numpy
-Scikit-Learn
-Matplotlib
-Google Colab
-Git hub

**##Workflow**
1.User interacts with the telegram chatbot
2.The chatbot collects menstrual cycle phase ,exercise , anxiety , triggers , coping methods data
3.Responses are automatically stored in Google Sheets
4. Data is cleaned and preprocessed using python
5. Exploratory data analysis (EDA) is performed to identify patterns.
6. Machine Learning techniques are used to analyze relationship between lifestyle factors , menstrual cycle phases and anxiety
7.Personalized insights and recommendations are generated

## Project Objective
The goal of this project is to analyze the relationship between menstruak cycle phases , lifestyle habits and anxiety levels . The system collects users responses through a telegram chatbot , stores then in a Google sheet automatically , and uses data analysis and machine learning techniques to identify patterns and generate personalized wellness insights
